# 비선상회 ERP v1.2

## 이번 버전
- Windows 관리자 화면(GUI) 추가
- 버튼으로 쿠팡 주문 처리 및 공급처별 발주서 생성
- 주문 파일, 발주 결과, 상품매핑표, 미매핑 목록, 처리 이력 바로 열기
- v1.1의 주문 누적관리 및 중복 발주 차단 유지
- 콘솔 실행 방식도 `콘솔실행.bat`로 유지

## 실행 방법
1. `data` 폴더에 처리할 쿠팡 주문 엑셀 1개를 넣습니다.
2. `실행.bat`를 더블클릭합니다.
3. 관리자 화면에서 `주문 처리 및 발주서 생성` 버튼을 누릅니다.
4. 신규 상품이 있다면 `상품 매핑표 열기`를 눌러 공급처와 내부상품 정보를 입력합니다.
5. 저장 후 다시 주문 처리 버튼을 누릅니다.

## 주의
- 처리 전 `output` 폴더의 엑셀 파일을 닫아야 합니다.
- `상품매핑표.xlsx`와 `주문처리이력.xlsx`는 운영 데이터이므로 삭제하지 마세요.
- 같은 주문은 주문처리이력을 기준으로 중복 발주되지 않습니다.
